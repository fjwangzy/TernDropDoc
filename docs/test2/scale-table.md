| **一、教学目标 (15分)** | **1. 学科育人价值 (5分)**      | **5分**：充分挖掘《日月潭》的写景文本价值，通过品读“风光秀丽”的具体描写，自然渗透对祖国宝岛台湾的热爱与向往之情，实现审美启蒙与爱国主义教育的有机融合。<br>**3-4分**：能提及课文的爱国情感与景色之美，但二者的结合较为生硬，或对“审美启蒙”这一育人点挖掘不深。<br>**0-2分**：仅停留在知识讲授层面，未能有效体现本文特有的文化传承与情感陶冶价值。                                                                                       |        |
|                  | **2. 教学目标制定 (5分)**      | **5分**：目标紧密对标第一学段课标要求，基于二年级学生认知，明确制定包含“识字写字”、“朗读感悟（重长句）”、“积累词语（四字词、比喻句）”及“初步感知文章脉络”等可观测、可评价的具体课时目标。<br>**3-4分**：目标基本符合课标与学情，但表述较为笼统（如“感受美景”），或对“语言积累与运用”这一核心要素强调不足。<br>**0-2分**：目标模糊、空泛，与《日月潭》具体的教学内容、重难点关联度低。                                                                 |        |
|                  | **3. 教学目标表述 (5分)**      | **5分**：目标表述规范、清晰，例如：“通过朗读与圈画，能找出描写日月潭清晨、中午、雨天景色的句子，并积累‘群山环绕’‘朦胧’等词语；能用自己的话简单介绍一种天气下的日月潭。”<br>**3-4分**：目标表述包含了“通过……能……”的格式，但行为动词不够具体，或预期成果不够明确。<br>**0-2分**：目标表述为“教师做什么”而非“学生学到什么”，或表述混乱，难以衡量。                                                                                  |        |
| **二、学习条件 (25分)** | **1. 教学环节设计 (5分)**      | **5分**：教学设计环节清晰（激趣导入→整体感知→品读赏析→总结拓展），逻辑连贯，每个环节任务明确（如“初识其名”、“概览其貌”），时间分配合理，符合从整体到局部、从理解到运用的认知规律。<br>**3-4分**：教学环节完整，但环节间的逻辑递进关系不够鲜明，或某个环节（如“整体感知”）流于形式，未能有效服务核心目标。<br>**0-2分**：教学环节设计零散、跳跃，或缺失关键环节（如缺少对文本语言的深入品读）。                                                              |        |
|                  | **2. 问题任务链设计 (12分)**    | **12分**：能设计富有思维梯度的核心问题链。如：①日月潭为什么叫这个名字？（理解特点）②它给你的整体印象是什么？（抓中心句）③课文写了哪些时候的景色？（理脉络）④晴天（晨、午）和雨天的美有什么不同？（对比品析）⑤假如你是小导游，怎么介绍？（迁移运用）。问题能引导学生进行信息提取、比较想象和语言表达。<br>**7-11分**：设计了主要问题，但问题的挑战性与层次性不足，或未能围绕“语言积累与运用”设计关键的语言实践任务（如换词比较、想象说话）。<br>**0-6分**：问题多为事实性、碎片化提问，缺乏主线，或脱离文本进行无效讨论。 |        |
|                  | **3. 信息技术融合 (8分)**      | **8分**：恰当运用地图软件定位台湾，精选展现日月潭“清晨湖光”、“中午晴空”、“雨天朦胧”不同意境的图片或短视频，有效辅助学生理解“倒映”、“朦胧”、“仙境”等难点，突出重点，增强体验。<br>**4-7分**：使用了多媒体资源，但资源与教学重难点结合不够紧密（如仅展示风光图），或信息技术应用方式单一，未能充分发挥其突破难点的作用。<br>**0-3分**：多媒体使用不当，干扰教学主线，或完全依赖课件，替代了必要的板书和师生对话。                                                    |        |
| **三、学习活动 (25分)** | **1. 教学活动设计 (5分)**      | **5分**：能设计并组织朗读（个人读、配乐读、对比读）、圈画关键词、想象画面、换词比较、小组交流、小导游介绍等多种学生活动，学生参与面广，教师适时范读、点拨，引导有效。<br>**3-4分**：学生活动以集体朗读和教师提问为主，活动形式较为单一，探究性、体验性活动不足，或教师指导过于主导，压制学生表达。<br>**0-2分**：课堂以教师讲授为主，学生被动听讲，缺乏有效的语文实践活动。                                                                            |        |
|                  | **2. 知识建构与迁移 (10分)**    | **10分**：能引导学生联系旧知（如“湖”与“潭”的比较，已学的简单比喻句），建构对“潭”、“环绕”、“朦胧”等新词的理解；通过梳理不同时段景色，感知“总分”的段落结构；最后通过“小导游”任务，促成积累词语（群山环绕、风光秀丽等）和句式（…像…）的迁移运用。<br>**6-9分**：注重了单点知识的教学，但未能有意识地帮助学生建立新旧知识间的联系，或迁移运用环节设计较为薄弱，未能有效检验学习成果。<br>**0-5分**：知识教学孤立，方法机械，学生未能形成结构化的认知，也无法运用所学解决新情境中的简单问题。             |        |
|                  | **3. 学生学习表现 (10分)**     | **10分**：学生在朗读中能努力读准字音、读通长句；在品读时能积极寻找关键词，大胆想象并描述画面（如“我好像看到了雾蒙蒙的湖面”）；在迁移任务中能尝试运用课文词语进行表达，学习兴趣浓厚。<br>**6-9分**：学生能跟随教师完成学习任务，但主动发现和提出问题的意识不强，思维参与度一般，情感体验不够深入。<br>**0-5分**：课堂气氛沉闷，学生参与度低，注意力涣散，未能表现出应有的学习状态。                                                                       |        |
| **四、学习指导 (15分)** | **1. 板书设计规范 (5分)**      | **5分**：主板书采用图文结合形式，用简笔画清晰呈现日潭、月潭、光华岛的位置关系，并用关键词（群山环绕、树木茂盛）和结构图（风光秀丽—清晨/中午/雨天—关键词）呈现文章脉络与重难点，布局合理，书写规范。<br>**3-4分**：板书有主要内容，但未能体现文章结构与内在联系，或缺乏简笔画等直观手段辅助理解难点，或书写潦草。<br>**0-2分**：板书随意、凌乱，或完全以课件投影代替板书，缺乏示范性与结构性。                                                                 |        |
|                  | **2. 学习习惯方法指导 (5分)**    | **5分**：能在课堂中相机指导学习方法，如：朗读时教给“边读边想象画面”的方法；理解词语时教给“联系上下文”、“结合插图”和“换词比较”的方法；阅读时教给“抓住中心句”把握文章大意的方法。<br>**3-4分**：有一些学习要求的指令（如“大声读”），但缺乏具体、可操作的方法指导，或指导未能贯穿学习过程。<br>**0-2分**：仅关注知识内容的传递，忽视对学生语文学习习惯与方法的系统指导。                                                                         |        |
|                  | **3. 教学反馈与生成资源利用 (5分)** | **5分**：能对学生的朗读、回答给予具体、有针对性的评价与点拨（如：“你读出了清晨的安静”）；能敏锐捕捉课堂生成（如学生提问“为什么水是碧绿的”），并灵活转化为教学资源，保护好奇心，引导探究。<br>**3-4分**：反馈以简单判断（如“好”、“不对”）为主，缺乏指导性；对于课堂生成问题，采取回避或简单告知答案的处理方式。<br>**0-2分**：忽视学生课堂表现，反馈缺失或消极，压制学生生成性问题。                                                                    |        |
| **五、学习评价 (15分)** | **1. 学习效果评价 (5分)**      | **5分**：能通过“脉络梳理填空”检测对文章内容与结构的整体把握；通过“画面描述任务”（用课文词语说其他雨景图）评价对关键语言的理解与迁移能力；评价贯穿教学始终，及时诊断目标达成度。<br>**3-4分**：有课堂练习或提问检测，但评价方式单一，或未能紧扣本课核心教学目标（如只检测字词，未检测对美景描写的理解与积累）。<br>**0-2分**：缺乏课堂即时评价环节，仅靠课后作业了解学情，无法及时调整教学。                                                                  |        |
|                  | **2. 作业设计质量 (5分)**      | **5分**：基础性作业（如抄写生字、背诵优美段落）经过精选；能设计实践性、综合性作业，如：画一画你心中的日月潭，并给家人用上课文中的词语做介绍；或收集关于台湾风景的图片资料。<br>**3-4分**：作业以常规的抄写、练习册题目为主，缺乏与课文特色（写景、优美语言）相结合的开放性、实践性作业设计。<br>**0-2分**：作业布置随意，量大质低，缺乏针对性和层次性。                                                                                      |        |
|                  | **3. 教学整体效果 (5分)**      | **5分**：学生能正确流利朗读课文，积累本课优美词句，对日月潭的“风光秀丽”有具体感知，并流露出对祖国山河的赞美之情和进一步了解台湾的愿望，课堂整体目标达成度高。<br>**3-4分**：基本完成知识性教学任务，但学生在情感共鸣、语言内化与运用方面表现不突出，课堂感染力一般。<br>**0-2分**：教学目标未达成，学生课后对课文内容印象模糊，缺乏学习获得感。                                                                                         |        |
| **六、特色创新 (5分)**  | **1. 教学创新实践 (5分)**      | **5分**：在教学实施中展现出鲜明特色与创意，如：巧妙运用简笔画板书将抽象方位与文本描述可视化；设计“晴雨对比阅读”活动，深度发展学生比较思维能力；创设“最美代言人”情境贯穿全课，极大激发学生表达欲。<br>**3-4分**：在教学某个环节有一定新意，但整体教学设计较为常规，特色不够突出。<br>**0-2分**：教学方式传统，按部就班，缺乏个性化的思考与设计。                                                                                        |        |
